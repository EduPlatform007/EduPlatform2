## УРОК 3: КЕСТЕЛЕР МЕН ФОРМАЛАР
==========================================
### Тест:

1. HTML-де кесте жасау үшін қандай тег қолданылады?
   Ответ: `<table>` (вариант b)

2. Кестенің тақырыбын қалай көрсетеміз?
   Ответ: `<caption>` (вариант a)

3. Кестенің баған тақырыбын қалай белгілейміз?
   Ответ: `<th>` (вариант c)

4. Форманы серверге жіберу үшін қандай атрибут қолданылады?
   Ответ: `action` (вариант d)

## УРОК 6: FLEXBOX ЖҮЙЕСІ
==========================================
### Тест:

1. Flexbox жүйесін қалай іске қосамыз?
   Ответ: `display: flex;` (вариант a)

2. Элементтерді контейнер ішінде орталықтандыру үшін қандай қасиет қолданылады?
   Ответ: `justify-content: center;` (вариант c)

3. flex-direction қасиетінің мақсаты қандай?
   Ответ: Элементтердің орналасу бағытын анықтау (вариант b)

4. Элементтерді тік бағытта орталықтандыру үшін қандай қасиет қолданылады?
   Ответ: `align-items: center;` (вариант a)

### Практическое задание: Создание flex-контейнера

```css
.container {
    display: flex;
    justify-content: center;
    flex-direction: column;
    align-items: center;
}
```

## УРОК 7: GRID ЖҮЙЕСІ
==========================================
### Тест:

1. Grid құрылымын қалай іске қосамыз?
   Ответ: `display: grid;` (вариант b)

2. Ұяшықтар арасындағы қашықтықты қалай орнатамыз?
   Ответ: `gap: 20px;` (вариант b)

3. Үш бірдей өлшемді баған жасау үшін қандай код қолданылады?
   Ответ: `grid-template-columns: repeat(3, 1fr);` (вариант a)

==========================================
## УРОК 8: АДАПТИВТІ ДИЗАЙН
==========================================
### Тест:

1. Адаптивті дизайн жасау үшін қандай CSS технология қолданылады?
   Ответ: `@media` сұраныстары (вариант c)

2. Мобильдік құрылғыларға арналған стильдерді қалай орнатуға болады?
   Ответ: `@media (max-width: 600px) { }` (вариант a)

3. viewport мета-тегінің мақсаты қандай?
   Ответ: Экран өлшемін басқару үшін (вариант b)

### Практическое задание: Создание адаптивных стилей

```css
@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
    
    .item {
        width: 100%;
    }
}
```

==========================================
### Тест (15 вопросов):

   Ответ: HyperText Markup Language (вариант a)

2. HTML құжатының негізгі элементі қандай?
   Ответ: `<html>` (вариант b)

3. Элементтерге қосымша ақпарат беру үшін не қолданылады?
   Ответ: Атрибуттар (вариант c)

7. CSS-ті HTML-ге қосудың ДҰРЫС ЕМЕС әдісі қандай?

10. Элементтерді контейнер ішінде орталықтандыру үшін қандай қасиет қолданылады?
    Ответ: `justify-content: center;` (вариант c)

11. Grid құрылымын қалай іске қосамыз?
    Ответ: `display: grid;` (вариант b)

12. Ұяшықтар арасындағы қашықтықты қалай орнатамыз?
    Ответ: `gap: 20px;` (вариант b)

13. Үш бірдей өлшемді баған жасау үшін қандай код қолданылады?
    Ответ: `grid-template-columns: repeat(3, 1fr);` (вариант a)

14. Адаптивті дизайн жасау үшін қандай CSS технология қолданылады?
    Ответ: `@media` сұраныстары (вариант c)

15. Мобильдік құрылғыларға арналған стильдерді қалай орнатуға болады?

### Практическое задание: Создание адаптивного сайта

```html
<!DOCTYPE html>
<html lang="kk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Адаптивті сайт</title>
  <style>
    body {
      font-family: Arial, sans-serif;
      margin: 0;
      padding: 0;
    }
    
    .container {
      display: flex;
      flex-wrap: wrap;
      max-width: 1200px;
      margin: 0 auto;
      padding: 20px;
    }
    
    .item {
      flex: 1 1 300px;
      margin: 10px;
      padding: 20px;
      background-color: #f0f0f0;
      border-radius: 5px;
    }
    
    @media (max-width: 768px) {
      .item {
        flex: 1 1 100%;
      }
    }
  </style>
</head>
<body>
  <div class="container">
    <div class="item">
      <h2>Бірінші бөлім</h2>
      <p>Мәтін мысалы</p>
    </div>
    <div class="item">
      <h2>Екінші бөлім</h2>
      <p>Мәтін мысалы</p>
    </div>
    <div class="item">
      <h2>Үшінші бөлім</h2>
      <p>Мәтін мысалы</p>
    </div>
  </div>
</body>
</html>
```
